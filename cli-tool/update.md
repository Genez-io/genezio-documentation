# Update genezio

Use the command below to install the latest version of genezio using you preferred package manager:

{% tabs %}
{% tab title="npm" %}
```
npm update genezio -g
```
{% endtab %}

{% tab title="yarn" %}
```
yarn global upgrade genezio 
```
{% endtab %}

{% tab title="pnpm" %}
```
pnpm update genezio -g
```
{% endtab %}
{% endtabs %}
